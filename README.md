# CSV Analyzer

A simple full-stack tool that converts raw CSV files into actionable insights.  
Built using core statistical concepts from my second-semester coursework.

## Features

- Column type detection  
- Summary statistics  
- Outlier detection  
- Histograms & basic charts  
- Drag-and-drop CSV upload
